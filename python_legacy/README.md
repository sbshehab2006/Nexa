# Project Nexa - Custom LLM

Welcome to Project Nexa. This project aims to build a custom Language Model (LLM) from scratch using PyTorch.

## Structure

- `model.py`: Contains the definition of the Nexa Transformer architecture.
- `train.py`: Script to train the model on a dataset.
- `inference.py`: Script to generate text using the trained model.
- `data/`: Directory for storing training data.

## Getting Started

1. Install dependencies:
   ```bash
   pip install -r requirements.txt
   ```

2. Prepare data (instructions to follow).

3. Train the model:
   ```bash
   python train.py
   ```

4. Generate text:
   ```bash
   python inference.py
   ```
